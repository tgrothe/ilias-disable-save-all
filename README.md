## ilias-disable-save-all

A simple Tampermonkey JavaScript script to disable the save all buttons in ILIAS to prevent accidental saving of unwanted changes.

## Install/update:

1. Install the Tampermonkey extension for your browser (e.g., Chrome).
2. Then, visit [ilias_script_tobias.user.js](ilias_script_tobias.user.js), and click the Raw button in the top right corner.
3. The script will be recognized, installed, and automatically updated by TM.
